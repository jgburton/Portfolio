About Me

I'm a Full Stack Web Developer and Musician originally from Derbyshire, currently based in London, having lived in Hong Kong for a couple of years. I studied Music at university, and recently completed the 3 month immersive web development course with General Assembly, having realised I'm passionate about creating and utilising technology to improve others' everyday lives.

Teaching, Music & Travel

After graduating from University I moved to Hong Kong to teach international students English alongside tutoring music privetely for almost two years. I've since continued to perform in a jazz orchestra amongst other smaller projects in London as a guitarist.

Web Development

JavaScript (ES5 and ES6, jQuery, AngularJS, Express, Node.js, EJS), HTML5, CSS3 (including SASS, SCSS, Bootstrap), Ruby (v2.3.1, Sinatra, Ruby on Rails v5+, ERB), databases (MongoDB, NoSQL, PostgreSQL, SQL), Gulp, Git, Heroku

Motivation

By focussing on tech within the creative, music and charity industries I’d like my work to contribute to impacting others’ everyday lives in a positive and subtle way. Alongside this I continue to work as a freelance musician and highly value the arts.

Tech Projects

Below are four applications I built during my time at General Assembly. I have included the Github links for more details of each project, including the technologies used.

Lick Library

Javascript Game
View Website View on Github

This was my first project on WDI, to understand the basic fundamentals of JavaScript. Lick Library is a simple piano memory game. The user can play the memory game or select the jam function to play the keyboard freely.

City Jazz

Node/Express Google Maps App
View Website View on Github

An authenticated app using the Google Maps and Yelp API’s alongside a city mapper integration. The app allows users to locate jazz venues in London, find out more information and navigate to them. Written in javaScript and jQuery, this was my first experience working with an external API.

Code Fund

MEAN Stack App
View Website View on Github

A team project, we built a RESTful app inspired by our experience so far in the tech world. CodeFund is a space to ask technical questions to fellow developers, with questioners donating to the charity chosen by the owner of the best answer. My teammates and I wanted to explore using tech for good; how to contribute to charity alongside providing a service.

Remote Composer

Rails/Angular
View Website View on Github

My final project on WDI using a Ruby of Rails backend and AngularJS front-end framework. Remote Composer is an open source music composition platform, for composers and artists who want to collaborate and develop musical ideas together. We decided to create a platform for composers to upload their unfinished ideas, with the intention of other composers being able to contribute to that 'idea' and result in a beautiful and collaborative composition.
